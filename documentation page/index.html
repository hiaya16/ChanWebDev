Introduction

Tailwind CSS is a utility-first CSS framework for rapidly building custom user interfaces. It is highly customizable and provides all the building blocks needed to create bespoke designs without any opinionated styles to override.

  • Installation

    To install Tailwind CSS, follow these steps:

    npm install tailwindcss

    Then create your tailwind.config.js file:

    npx tailwindcss init

    Usage

    Here's how to use Tailwind CSS in your project:

    
/* Add the Tailwind directives to your CSS */
@tailwind base;
@tailwind components;
@tailwind utilities;

    Then you can start using Tailwind's utility classes in your HTML:

    
<h1 class="text-3xl font-bold underline">
  Hello world!
</h1>

    API

    The Tailwind CSS API provides various configuration options. You can customize the default theme, extend the default utility classes, and configure plugins. The configuration file is a JavaScript file where you define your customizations:

    
module.exports = {
  theme: {
    extend: {
      colors: {
        'custom-blue': '#1fb6ff',
        'custom-pink': '#ff49db',
      }
    }
  }
}

    Examples

    Here are some examples of how to use Tailwind CSS:

    
// Example 1: Button
<button class="bg-blue-500 hover:bg-blue-700 text-white font-bold py-2 px-4 rounded">
  Button
</button>

// Example 2: Card
<div class="max-w-sm rounded overflow-hidden shadow-lg">
  <img class="w-full" src="/img/card-top.jpg" alt="Sunset in the mountains">
  <div class="px-6 py-4">
    <div class="font-bold text-xl mb-2">The Coldest Sunset</div>
    <p class="text-gray-700 text-base">
      Lorem ipsum dolor sit amet, consectetur adipisicing elit.
    </p>
  </div>
</div>
